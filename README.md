# hello-nrepl-middleware

[nREPL](https://github.com/nrepl/nrepl) Middleware Example

## Usage

starting test nREPL server
```
$ lein repl
user=> (go)
```

connect to test nREPL server
```
$ nc localhost 12345
d2:op5:helloe

d5:hello5:world7:session36:YOUR_SESSION_ID:statusl4:doneee
```
